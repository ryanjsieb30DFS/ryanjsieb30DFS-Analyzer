# PGA Course References

Per-course markdown files cataloguing physical profile + comparable courses. Used to provide cross-slate context when reviewing or planning a golf slate. **Shared by both PGA slugs** (`pga_classic` and `pga_rd4_sd`) — same physical courses, one knowledge base.

## Convention

- One `.md` file per course, named `<course_slug>.md` (e.g. `muirfield_village.md`)
- Course data comes from the slate's articles and the user — saved here as a cumulative knowledge base
- Each file has: physical profile (length, par, agronomy, key skill tests — driving, approach buckets, scrambling), direct comparables, strategic notes (accumulates date-stamped "Per-slate observation" lines via the post-autopsy review)
- A file created mid-slate from articles alone is marked `**UNVERIFIED — built from this slate's articles only**` until a post-autopsy review confirms it

## Adding a new course

The Pre-flight ritual creates a stub automatically when a slate features a course not yet here; the post-autopsy review then appends what the slate actually proved or disproved about the course.
